## 一、入口与流程

### 1.1 状态机新增

```
menu → charSelect → skinSelect → mapSelect → buildSelect → playing
                      ↑ 新增状态
```

在 `charSelect`（角色选择）界面，选中职业后，添加「换肤」按钮进入 `skinSelect` 状态。

### 1.2 入口位置

在 `drawCharSelect()` 中，每个职业卡片右侧添加一个「🎨 换肤」按钮：
- 位置：职业卡片内右上角，`W/2 + 120, by + 8`
- 尺寸：60 x 24 px
- 颜色：`#2a2a4e` 背景，`#ffd700` 文字
- 点击后：`state = 'skinSelect'`，并记录当前选择的职业

### 2.2 响应式适配

| 屏幕宽度 | 每行皮肤数 | 卡片尺寸 | 预览区高度 |
|----------|-----------|---------|-----------|
| W >= 700 | 6 | 70×90 | 220px |
| 500 <= W < 700 | 5 | 65×85 | 180px |
| W < 500 | 4 | 60×80 | 150px |

计算方式：
```javascript
var cols = W >= 700 ? 6 : (W >= 500 ? 5 : 4);
var cardW = Math.floor((W - 80) / cols); // 左右各留 20px + 间距
var cardH = cardW * 1.28; // 宽高比约 1:1.28
```

## 三、预览区规格

### 3.1 大图预览

| 属性 | 值 |
|------|-----|
| 区域 | 居中，宽 `W * 0.5`，高 `180~220px` |
| 背景 | 径向渐变：中心 = 皮肤品质色 alpha 0.15，边缘 = `#0a0a12` |
| 角色大小 | `radius = 40`（约为游戏内的 3x） |
| 旋转动画 | `facingAngle = gameTime * 1.5`（每 4.2 秒一圈） |
| 特效渲染 | 全部启用（光环/粒子/描边），使用 `drawCharacterSprite()` |
| 皮肤数据 | `getSkinData(selectedSkinId)` |

### 3.2 品质背景色

```javascript
var tierBg = {
  C: '#333333', B: '#1a2a44', A: '#2a1a3a', S: '#3a2a10'
};
var tierBorder = {
  C: '#aaaaaa', B: '#4488ff', A: '#cc66ff', S: '#ffaa00'
};
var tierStars = { C: 1, B: 2, A: 3, S: 4 };
```

## 四、皮肤���片规格

### 4.1 单个卡片

```
┌─────────────┐  ← 品质边框 (2px, tierBorder色)
│             │
│   角色缩略   │  ← radius = 14, 带配色但不带粒子/光环
│             │
│  皮肤名      │  ← 12px, #fff, 单行截断
│  ★★☆☆      │  ← 品质星星, tierBorder色
└─────────────┘
```

| 属性 | 值 |
|------|-----|
| 默认尺寸 | 70 x 90 px（响应式调整） |
| 间距 | 8px |
| 圆角 | 6px |
| 背景 | `tierBg[skin.tier]`，选中时亮度+20% |
| 边框 | `tierBorder[skin.tier]`，选中时宽度 3px + shadowBlur 6 |
| 角色预览 | `drawCharacterSprite(cx, cy, 14, selectedClass, 0, { skinId })` |
| 皮肤名 | `12px Arial`，居中，单行，超出省略 |
| 品质星星 | 每颗 `8px`，间距 2px，颜色 `tierBorder` |
| 未拥有 | 叠加半透明黑色遮罩 `rgba(0,0,0,0.6)` + 🔒 图标 |
| 触摸热区 | 整个卡片，最小 48px 宽（已满足） |

### 4.2 排序规则

1. 已拥有在前，未拥有在后
2. 同组内按品质降序：S > A > B > C
3. 同品质按 skinId 字母序

### 4.3 职业过滤

只显示 `applicableClasses` 包含当前选中��业的皮肤。

## 六、信息栏规格

位于预览区下方，显示选中皮肤的详细信息：

```javascript
// 皮肤名 + 品质
ctx.font = 'bold 16px Arial'; ctx.fillStyle = tierBorder[skin.tier];
ctx.fillText(skin.name, W/2 - 120, 295);

// 品质文字
ctx.font = '13px Arial'; ctx.fillStyle = tierBorder[skin.tier];
ctx.fillText('★'.repeat(tierStars[skin.tier]) + '☆'.repeat(4 - tierStars[skin.tier]) + ' ' + tierName, W/2 + 60, 295);

// 描述
ctx.font = '12px Arial'; ctx.fillStyle = '#aaa';
ctx.fillText(skin.description, W/2 - 120, 312);
```

品质中文名映射：
```javascript
var tierNames = { C: '普通', B: '精良', A: '史诗', S: '传说' };
```

## 七、滚动交互

皮肤网格区域支持触摸/鼠标滚动：

```javascript
var skinScrollY = 0;        // 当前滚动偏移
var skinScrollMax = 0;       // 最大滚动值 (根据行数计算)
var skinScrollVelocity = 0;  // 惯性滚动速度

// 触摸拖拽滚动
touchmove → skinScrollY -= deltaY
touchend  → skinScrollVelocity = lastDeltaY * 0.8 (惯性)

// 鼠标滚轮
wheel → skinScrollY -= ev.deltaY * 0.5

// 裁剪区域 (防止溢出)
ctx.save();
ctx.beginPath();
ctx.rect(20, 320, W - 40, 200); // 皮肤网格可视区
ctx.clip();
// ... 绘制皮肤卡片 ...
ctx.restore();
```

## 八、动效与转场

### 8.1 进入动画

皮肤卡片从底部渐入，每张延迟 30ms：
```javascript
var enterProgress = Math.min(1, (Date.now() - enterTime) / 300); // 300ms
var cardOffsetY = (1 - enterProgress) * 40; // 从下方40px滑入
var cardAlpha = enterProgress;
```

### 8.2 选中动画

选中卡片有 120ms 的缩放弹跳：
```javascript
var selectScale = 1.0 + 0.08 * Math.sin(selectTimer * Math.PI); // 弹跳
```

### 8.3 预览区角色切换

切换皮肤时，预览角色有 200ms 的淡出→淡入：
```javascript
previewAlpha = Math.abs(Math.cos(switchTimer / 200 * Math.PI));
```

### 9.3 预览 Canvas

- 复用 `drawCharacterSprite()` 函数
- 背景：径向渐变，中心品质色
- 角色 `radius = 30`，自动旋转
- 切换职业时实时更新（三个按钮切换 warrior/mage/scout）
- 特效参数修改后实时刷新预览

### 9.4 保存

同其他 Tab，点击「保存」调用 `POST /api/editor/skins`，数据格式为 `data/skins.json` 的完整内容。
