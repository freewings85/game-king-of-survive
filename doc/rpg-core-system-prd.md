# RPG 核心系统设计 PRD

## 背景

当前游戏的所有数值（角色 HP、攻击力、怪物属性、技能效果）都是硬编码的魔法数字，没有统一的数值体系。结果是：
- 加新技能就改代码，平衡靠猜
- 怪物强度靠 wave 线性缩放，缺乏层次感
- 角色三职业（战士/法师/斥候）的差异只是几个写死的数字
- 无法支撑大秘境等需要精细数值设计的玩法

**目标**：建立一套数据驱动的 RPG 数值体系，所有战斗相关数字从属性公式推导，支持编辑器可视化配置。

## 一、角色属性系统

### 1.1 四维基础属性

| 属性 | 缩写 | 战斗效果 | 公式 |
|------|------|---------|------|
| **智力** (Intelligence) | INT | 技能伤害、冷却缩减、法力上限 | 技能伤害加成 = INT × 0.8%；冷却缩减 = INT × 0.3%（上限40%）；法力上限 = 50 + INT × 2 |
| **力量** (Strength) | STR | 物理伤害、最大生命值、护甲 | 物理伤害加成 = STR × 1.0%；HP加成 = STR × 3；护甲 = STR × 0.5 |
| **敏捷** (Agility) | AGI | 移动速度、攻击速度、闪避率 | 移速加成 = AGI × 0.5%；攻速加成 = AGI × 0.4%；闪避率 = AGI × 0.15%（上限30%） |
| **耐力** (Stamina) | STA | 生命回复、抗性、受控缩减 | HP回复 = STA × 0.1/s；抗性 = STA × 0.4%（上限50%）；受控时间缩减 = STA × 0.3% |

### 1.2 二级战斗属性（从四维推导）

| 二级属性 | 计算公式 | 说明 |
|---------|---------|------|
| 最大HP | `baseHP + STR × 3 + characterLevel × 5` | 基础值来自职业 |
| 物理攻击力 | `baseATK × (1 + STR × 0.01)` | 百分比加成 |
| 技能威力 | `baseSkillDMG × (1 + INT × 0.008)` | 技能面板伤害 × 智力加成 |
| 攻击速度 | `baseAtkSpeed × (1 + AGI × 0.004)` | 影响自动攻击频率 |
| 移动速度 | `baseSpeed × (1 + AGI × 0.005)` | 上限 400 |
| 冷却缩减 | `INT × 0.003` | 上限 0.4（40%） |
| 暴击率 | `baseCrit + AGI × 0.001` | 上限 0.5（50%） |
| 暴击伤害 | `1.5 + STR × 0.005` | 基础 150% |
| 护甲 | `baseArmor + STR × 0.5` | 减伤 = 护甲/(护甲+100) |
| 抗性 | `STA × 0.004` | 减少技能伤害百分比 |
| HP回复 | `baseRegen + STA × 0.1` | 每秒回复量 |
| 闪避率 | `baseDodge + AGI × 0.0015` | 上限 0.3 |

### 1.3 伤害计算公式

```
最终伤害 = 基础伤害 × 属性加成 × (1 - 护甲减免) × (1 - 抗性) × 暴击倍率 × 随机浮动(0.9~1.1)

其中：
- 护甲减免 = 目标护甲 / (目标护甲 + 100)
- 暴击倍率 = if (rand < 暴击率) then 暴击伤害 else 1.0
```

## 二、角色等级与经验系统

### 2.1 角色等级（局外永久等级）

区别于现有的局内等级（每局从1级开始），新增**账号等级**：

| 等级范围 | 升级所需经验 | 公式 |
|---------|------------|------|
| 1-10 | 100 ~ 550 | `level × 50 + 50` |
| 11-30 | 660 ~ 2450 | `level × 60 + 60` |
| 31-50 | 2700 ~ 6250 | `level × 80 + 230` |
| 51+ | 6500+ | `level × 100 + 1000` |

**升级奖励**：
- 每级获得 **3 个属性点**（自由分配到 INT/STR/AGI/STA）
- 每 5 级解锁 1 个技能槽位
- 每 10 级获得 1 个天赋点

### 2.2 局内等级（保持现有机制）

局内等级继续使用现有公式 `15 + level*12 + level^1.8*3`，但效果改为：
- 每级获得 **1 个临时属性点**（仅本局有效）
- 每级从 3 个随机技能中选 1 个（保持现有）

### 2.3 经验获取来源

| 来源 | 经验值 | 说明 |
|------|-------|------|
| 击杀普通怪 | 怪物等级 × 5 | 等级差修正 |
| 击杀精英怪 | 怪物等级 × 15 | — |
| 击杀Boss | 怪物等级 × 50 | — |
| 击杀玩家 | 对方等级 × 20 | PvP |
| 完成一局 | 100 + 存活时间 × 2 | 基础保底 |
| 获得胜利 | 500 | 额外奖励 |

**等级差修正**（防刷低级怪）：
```
if (怪物等级 < 角色等级 - 5):
    经验 × max(0.1, 1 - (角色等级 - 怪物等级 - 5) × 0.2)
if (怪物等级 > 角色等级 + 3):
    经验 × 1.5  // 挑战高级怪有奖励
```

## 三、职业系统

### 3.1 职业定义（重构现有 CLASS_DEFS）

保留三职业，但属性化重新定义：

| 职业 | 基础HP | 基础ATK | 基础速度 | 初始属性分配 | 被动技能 |
|------|--------|---------|---------|------------|---------|
| **战士** | 250 | 12 | 160 | STR 8, STA 6, AGI 3, INT 3 | 荆棘护甲：受到近战伤害时反弹 `STR × 0.3` 伤害 |
| **法师** | 120 | 8 | 180 | INT 10, AGI 4, STA 3, STR 3 | 奥术延伸：技能射程 +50%，技能伤害额外 +`INT × 0.2%` |
| **斥候** | 150 | 15 | 240 | AGI 8, STR 4, INT 3, STA 5 | 闪避本能：闪避率额外 +`AGI × 0.1%`，闪避后下次攻击必暴击 |

### 3.2 属性成长系数

不同职业升级时，每点属性的效果略有不同（鼓励但不强制本职业方向）：

| 属性 | 战士系数 | 法师系数 | 斥候系数 |
|------|---------|---------|---------|
| STR | 1.2 | 0.8 | 1.0 |
| INT | 0.8 | 1.2 | 0.9 |
| AGI | 0.9 | 1.0 | 1.2 |
| STA | 1.1 | 0.9 | 1.0 |

> 例如：战士加 1 点 STR，实际获得 1.2 × 标准效果

## 四、怪物系统

### 4.1 怪物等级

怪物不再只由 wave 决定强度，而是有**独立等级**：

```
怪物等级 = 区域基础等级 + wave偏移 + 随机浮动(-1~+1)
```

| 区域 | 基础等级 | 怪物类型 |
|------|---------|---------|
| 新手区（外围） | 1-3 | normal, swarm |
| 东部/西部区 | 3-6 | normal, fast, ranged |
| 危险区（中心） | 5-10 | tank, miniBoss, boss |
| 大秘境 | 钥石层数×2 | 全类型 + 词缀 |

### 4.2 怪物属性计算

```
怪物HP = 基础HP × (1 + (怪物等级 - 1) × 0.15)
怪物攻击 = 基础ATK × (1 + (怪物等级 - 1) × 0.12)
怪物速度 = 基础Speed × (1 + (怪物等级 - 1) × 0.03)
掉落经验 = 基础XP × (1 + (怪物等级 - 1) × 0.1)
```

### 4.3 怪物类型数据表

| 类型 | 基础HP | 基础ATK | 基础Speed | 基础XP | 特征 |
|------|--------|---------|-----------|--------|------|
| normal | 20 | 12 | 75 | 10 | 标准近战 |
| fast | 10 | 8 | 155 | 8 | 高速低血 |
| tank | 60 | 14 | 50 | 20 | 肉盾 |
| ranged | 18 | 10 | 56 | 12 | 远程攻击 |
| swarm | 6 | 5 | 130 | 4 | 成群出现 |
| miniBoss | 150 | 18 | 65 | 45 | 精英，有技能 |
| boss | 300 | 22 | 38 | 80 | Boss，多阶段 |
| treasure | 30 | 0 | 190 | 50 | 逃跑型，高金币 |

> 这些数据应存储在 JSON 配置文件中，代码通过读取配置来生成怪物

## 五、技能系统

### 5.1 技能数据结构

每个技能是一条数据记录，而非硬编码逻辑：

```json
{
  "id": "fireball",
  "name": "火球术",
  "icon": "🔥",
  "color": "#f80",
  "type": "projectile",           // projectile | aoe | buff | summon | passive
  "damageType": "magic",          // physical | magic
  "baseDamage": 50,
  "scalingAttribute": "INT",      // 哪个属性影响伤害
  "scalingRatio": 3.2,            // 每点属性增加的伤害
  "cooldown": 2.0,                // 秒
  "manaCost": 15,
  "range": 400,
  "aoeRadius": 0,                 // 0 = 单体
  "projectileSpeed": 500,
  "projectileCount": 1,
  "pierce": false,
  "maxLevel": 5,
  "levelScaling": {
    "damage": 0.2,                // 每级伤害 +20%
    "cooldown": -0.1,             // 每级冷却 -10%
    "range": 0.05                 // 每级射程 +5%
  },
  "description": "发射火球，造成 {damage} 点火焰伤害",
  "unlockLevel": 1,
  "classRestriction": null        // null = 所有职业可用, "mage" = 法师专属
}
```

### 5.2 技能类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **projectile** | 发射弹幕 | 火球、冰箭、闪电链 |
| **aoe** | 范围伤害 | 地震、冰冻光环、火焰轨迹 |
| **buff** | 增益效果 | 攻击力提升、护盾、加速 |
| **summon** | 召唤物 | 环绕护卫、荆棘光环 |
| **passive** | 被动效果 | 暴击、吸血、经验磁铁 |

## 七、编辑器

### 7.1 技能编辑器

独立页面 `editor/skill-editor.html`：
- 表格形式编辑所有技能参数
- 实时预览技能效果（Canvas 小窗口）
- 导出为 `data/skills.json`
- 伤害模拟器：输入属性值，显示不同等级的伤害输出
- 技能进化关系图

### 7.2 地图编辑器

独立页面 `editor/map-editor.html`：
- 可视化拖拽编辑地图
- 放置区域（设置怪物等级范围、刷怪速率、怪物类型）
- 放置障碍物和地形元素
- 设置出生点
- 导出为 `data/maps/{mapId}.json`
- 预览模式：在编辑器内试玩

### 7.3 怪物编辑器

内嵌在地图编辑器中：
- 编辑怪物类型的基础属性
- 预览不同等级下的属性值
- 导出为 `data/monsters.json`

## 十一、验收标准

1. **属性公式正确**：修改属性值后，战斗数值按公式线性变化，可通过模拟器验证
2. **数据驱动**：修改 JSON 文件中的数值后，不改代码即可生效
3. **编辑器可用**：可以通过编辑器创建新技能/新地图，导出 JSON 后游戏可加载
4. **向后兼容**：现有的 3 职业、18 技能、9 地图在新系统下功能一致
5. **平衡验证**：8 玩家模拟器跑 16 局，各职业胜率在 25%~40% 之间
