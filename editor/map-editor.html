地图编辑器 - Map Editor

操作引导：

  • 点击"添加区域"创建新区域，在 Canvas 上显示
  • 拖拽区域可移动位置，拖拽边缘可调整大小
  • 点击"放置障碍物"后在 Canvas 上点击放置
  • 点击"放置出生点"后在 Canvas 上点击设置
  • 使用 Ctrl+Z 撤销、Ctrl+Y 重做
  • 导出的 JSON 可直接用于游戏地图配置

参数说明：

  • Lv: 区域内怪物等级范围(最小-最大)
  • Rate: 怪物刷新速率(1.0=正常)
  • 形状: 地图边界形状(rect/circle/hex等)
  • 段位要求: 进入此地图所需的最低段位
