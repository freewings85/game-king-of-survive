# Character Sprite Credits

## Current status: SPEC-COMPLIANT PLACEHOLDERS

The three sprite sheets in this directory (`lpc_warrior.png`, `lpc_mage.png`,
`lpc_scout.png`) are **procedurally generated placeholders** that follow the
exact LPC (Liberated Pixel Cup) sprite sheet specification:

- **Frame size:** 64×64
- **Sheet size:** 832×1344 (13 cols × 21 rows)
- **Direction order:** Up / Left / Down / Right (LPC standard)
- **Row layout:**
  - rows  0–3:  spellcast (7 frames per direction)
  - rows  4–7:  thrust   (8 frames)
  - rows  8–11: walk     (9 frames, frame 0 = idle pose)
  - rows 12–15: slash    (6 frames)
  - rows 16–19: shoot    (13 frames)
  - row  20:    hurt     (6 frames, down only)

These placeholders allow the engine code (`SpriteAnimator`, `drawCharacterSprite`,
editor preview, sprites.json wiring) to be built and tested **immediately**.

## Swap-in plan for real LPC art

When real LPC assets are downloaded from
<https://opengameart.org/content/liberated-pixel-cup-lpc-base-assets-sprites-map-tiles>
and the Universal LPC Spritesheet Character Generator
(<https://sanderfrenken.github.io/Universal-LPC-Spritesheet-Character-Generator/>),
**simply overwrite the three PNG files**. No code changes required — sheet
spec is locked.

Composition recipes (per GameDesigner v1 doc):

| Class    | Body            | Armor                  | Weapon  | Hair / Head     | Attack anim |
|----------|-----------------|------------------------|---------|------------------|-------------|
| warrior  | male, light     | plate (chest+legs)     | longsword | helmet         | slash       |
| mage     | male, light     | long robe (purple)     | staff   | hood up          | spellcast   |
| scout    | female, tan     | leather + green cloak  | bow     | short hair       | shoot       |

## Required attribution (when real LPC art is in use)

LPC assets are dual-licensed under **CC-BY-SA 3.0** and **GPL 3.0**.
Once real assets replace the placeholders, this file MUST list every
contributing artist for each component used. Example format:

```
Body (male):     Stephen Challener (Redshrike), hosted by OpenGameArt.org
Hair:            bluecarrot16
Plate armor:     Matthew Krohn (makrohn)
Longsword:       ... etc.
```

The Universal LPC Spritesheet Character Generator outputs a per-character
credits text file when you export — paste it into this file before shipping.
