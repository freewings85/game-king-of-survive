## Swap-in plan for real LPC art

When real LPC terrain assets are downloaded from:
- <https://opengameart.org/content/lpc-terrains> (by Lanea Zimmerman / Sharm)
- <https://opengameart.org/content/liberated-pixel-cup-lpc-base-assets-sprites-map-tiles>

**simply overwrite `lpc_terrain.png`** with a 384Г—256 sheet using the same
6-column biome order above. No code changes are required вҖ” `tileset.json`
holds the contract.

If you want to reorder or add biomes, update both `tileset.json` and the
`TERRAIN_BIOMES` constant in `demo/survivor.html` (biome `id` вҶ’ `col`).
