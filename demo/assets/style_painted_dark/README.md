# style_painted_dark

**风格定位：** 暗黑破坏神 1/2 风。深棕土色 + 暗红 + 骨白。强 vignette，粗轮廓，血锈点缀。
**适用场景：** 暗黑地牢、末世沙场、高压生存氛围。

## 文件
- `terrain.svg` — 石板 + 苔藓过渡 + 血迹
- `character_barbarian.svg` — 野蛮人（大肩甲 + 双手斧 + 披风，远看宽大轮廓辨识度强）
- `monster_skeleton.svg` — 骷髅（红眼 + 破袍 + 锈刀）
- `ui_frame.svg` — 铜钉皮革 + 骷髅徽章 + 铁角板
