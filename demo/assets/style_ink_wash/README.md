# style_ink_wash

**风格定位：** 东方水墨风。米色宣纸底 + 黑灰墨色 + 朱红印章点睛。差异化候选（如想和市面同类吃鸡拉开距离）。
**适用场景：** 武士/忍者主题、极简美学、品牌辨识度强。
**风险：** 辨识度高但受众窄；远距离看战斗场面可能对比度不够，需要谨慎调色。
