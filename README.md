# 生存之王 (King of Survive)

多人联机竞技生存游戏 + Java Spring Boot 后端。

## 游戏概述

玩家注册登录后进入游戏，选择单排或组队模式。打怪升级获取技能，消灭敌对阵营所有玩家即为胜利。

## 核心玩法

### 游戏模式

- **组队模式 (4v4)**：4 人一队（绿色 vs 红色），两队对抗
  - 同队友方不可攻击（友伤免疫）
  - 当己方全部 4 人阵亡，团队判负
  - 玩家阵亡后可观战队友
  - **团队胜利后触发魔山 Boss 战**：全队一起挑战，击杀获得 500 金币 + 皮肤碎片
- **单排模式 (8人混战)**：8 名玩家各自为战，最后存活者获胜（吃鸡）
- **大秘境 (PvE 挑战)**：1-4 人组队挑战，层数无上限的 PvE 极限模式

### 大秘境系统

类似暗黑破坏神大秘境 / 魔兽世界史诗钥石，纯 PvE 极限挑战模式。

#### 进入条件

- **秘境钥石**：通关竞技模式（吃鸡 / 团队胜利）掉落对应层数的钥石
  - 首次获取：完成任意一局竞技模式 → 获得 1 层钥石
  - 通关升级：在时限内通关 → 钥石层数 +1~3（视剩余时间）
  - 超时通关：通关但超时 → 钥石层数不变
  - 未通关：钥石层数 -1（最低 1 层）
- **组队**：1-4 人，无阵营对抗，全员协作

#### 核心机制

```
┌──────────────────────────────────────────────────┐
│              大秘境 · 第 N 层                      │
│                                                    │
│  进度条: ████████░░░░░░░░░░░░ 42%    ⏱ 2:31/4:00  │
│                                                    │
│  词缀: [强化] [易爆] [残暴]                         │
│                                                    │
│  击杀怪物 → 进度+%                                  │
│  进度 100% → 召唤秘境守护者（Boss）                  │
│  击杀 Boss → 通关                                   │
└──────────────────────────────────────────────────┘
```

**进度条**：击杀怪物填充进度（小怪 1-3%，精英 5-8%，到 100% 召唤 Boss）

**时间限制**：

| 层数 | 时限 | 怪物强度 |
|------|------|---------|
| 1-5 | 5 分钟 | HP ×1~2, DMG ×1~1.5 |
| 6-10 | 4 分钟 | HP ×2~4, DMG ×1.5~2.5 |
| 11-15 | 3.5 分钟 | HP ×4~8, DMG ×2.5~4 |
| 16-20 | 3 分钟 | HP ×8~16, DMG ×4~6 |
| 21+ | 3 分钟 | 每层 +15% HP/DMG，无上限 |

#### 词缀系统

层数越高，叠加越多词缀（类似魔兽 M+ 词缀）：

| 层数 | 词缀数 | 可选词缀 |
|------|--------|---------|
| 1-3 | 0 | 无词缀，纯净模式 |
| 4-6 | 1 | **强化**（精英怪 HP+40%）/ **急速**（怪物移速+30%） |
| 7-9 | 2 | 追加 **易爆**（怪物死亡留爆炸球）/ **崩裂**（怪物死亡释放治疗波） |
| 10-12 | 3 | 追加 **残暴**（怪物暴击率+20%）/ **感染**（受伤中毒持续掉血） |
| 13+ | 3+1 | 追加 **暴君**（Boss 获得额外技能阶段）|

#### 秘境守护者（Boss）

进度 100% 后召唤，不同于竞技模式的魔山 Boss，秘境守护者有独立 Boss 池：

| Boss | 特征 | 技能 |
|------|------|------|
| 虚空行者 | 高速移动 | 瞬移突袭 + 暗影弹幕 |
| 熔岩巨兽 | 超高血量 | 地面岩浆 + 践踏震荡 |
| 寒冰女皇 | 控制型 | 冰冻光环 + 召唤冰刺 |
| 雷霆之王 | 爆发型 | 连锁闪电 + 雷暴区域 |
| 深渊领主 | 13层+专属 | 融合前4个Boss技能 |

#### 奖励系统

| 通关条件 | 奖励 |
|----------|------|
| 超时通关 | 基础金币 × 层数 |
| 限时通关 | 金币 ×2 + 钥石升级 +1 |
| 快速通关（<50% 时限） | 金币 ×3 + 钥石升级 +2 + 皮肤碎片 |
| 极速通关（<30% 时限） | 金币 ×5 + 钥石升级 +3 + 限定皮肤碎片 |
| 首次通关该层 | 额外成就 + 称号解锁 |

**专属掉落**：
- 5 层+：秘境套装碎片（集齐 = 局内属性加成）
- 10 层+：稀有皮肤碎片
- 15 层+：传说皮肤碎片
- 20 层+：赛季限定皮肤直接掉落

#### 大秘境排行榜

- 按层数排名：谁打到的最高层
- 按速度排名：同层谁最快通关
- 赛季重置：每周排行榜重置，赛季结束发放排名奖励

#### 与竞技模式的关系

```
竞技模式（PvP）──胜利──→ 获得秘境钥石
       │                      │
       │                      ▼
       │              大秘境（PvE）──通关──→ 高级装饰/称号
       │                      │
       │                      ▼
       └── 永久强化 ◄── 金币/经验/皮肤 ── 反哺竞技实力
```

竞技模式提供钥石入场券，大秘境提供独占奖励（皮肤/称号），两个模式互相驱动玩家循环参与。

### 完整游戏流程

1. **注册/登录** → 访问 `http://localhost:8080`，输入昵称注册
2. **自动签到** → 每日签到奖励（7天循环：金币/经验/皮肤碎片）
3. **选择模式** → 单排（8人混战）/ 组队（4v4）/ 大秘境（PvE 挑战）
4. **选择角色** → 3 个职业（战士/法师/射手）各有被动技能
5. **选择地图** → 按段位解锁不同地图（绿野平原/暗影森林/熔岩峡谷等）
6. **PvE 阶段** → 打怪升级，选择技能，逐步变强
7. **PvP 对抗** → 消灭所有敌对玩家
   - 30 秒后若仅剩 ≤3 人存活，触发风暴缩圈逼迫决战
8. **胜利/失败** →
   - 组队胜利：触发魔山 Boss 战，击杀获得大量奖励
   - 单排胜利：大吉大利，今晚吃鸡
   - 失败：获得击杀和波次奖励金币
   - 竞技胜利 → 获得秘境钥石（可进入大秘境）
9. **积分结算** → 段位升降
10. **大秘境（可选）** → 使用钥石进入 → 限时清怪 → 击杀Boss → 获得独占奖励

### 战斗机制

- **自动攻击**：自动攻击最近的敌人（怪物和敌方玩家）
- **鼠标/触屏控制**：移动方向
- **技能系统**：升级时从 3 个随机技能中选 1 个
- **新手保护**：开局 5 秒 50% 减伤
- **复活机制**：首次死亡可花费金币原地复活（3 秒无敌 + 清除附近怪物）
- **Bot AI**：
  - 追踪范围 800px，攻击范围 450px
  - 18 伤害 / 0.7 秒冷却
  - 智能走位：躲避风暴、追杀弱敌

### 最终 Boss "魔山"

- **组队模式专属**：团队胜利后自动召唤
- 2000+ HP，随游戏时长持续增强（+10%/分钟）
- 多阶段攻击模式
- 击杀奖励：500 金币 + 皮肤碎片

### 地图系统

- 5 张地图：绿野平原、暗影森林、熔岩峡谷、冰霜荒原、迷雾沼泽
- 不同形状（矩形/六边形/圆形/菱形）
- 世界大小 2400×2400 像素，相机跟随玩家
- 不同区域刷怪速度和类型不同（中心高危高奖、边缘安全低奖）
- 风暴缩圈：30 秒后当存活人数 ≤3 时激活，60→100 px/s 加速收缩

### 角色成长

- **局内成长**：打怪升级 → 选技能 → 变强（30+ 种技能）
- **局外成长**：积分累计 → 段位提升（青铜→白银→黄金→铂金→钻石→大师）
- **永久强化**：金币购买体质/力量/速度/暴击等永久加成
- **皮肤系统**：纯装饰，通过签到/魔山掉落/碎片合成获取

### 后端 API

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/players` | 注册玩家 |
| GET | `/api/players/{id}` | 查询玩家信息 |
| PUT | `/api/players/{id}/equip-skin` | 装备皮肤 |
| POST | `/api/players/{id}/claim-daily-reward` | 领取每日签到奖励 |
| POST | `/api/rooms` | 创建房间 |
| GET | `/api/rooms` | 查看房间列表 |
| POST | `/api/rooms/{id}/join` | 加入房间 |
| POST | `/api/rooms/{id}/ready` | 准备/取消准备 |
| PUT | `/api/rooms/{id}/map` | 选择地图 |
| POST | `/api/rooms/{id}/start` | 开始游戏 |
| POST | `/api/scores/calculate` | 积分结算 |
| GET | `/api/scores/rank/{playerId}` | 查询段位排名 |
| GET | `/api/scores/leaderboard` | 击杀排行榜（支持?limit=N） |
| GET | `/api/skins` | 所有可用皮肤列表 |
| GET | `/api/skins/{skinId}` | 查询单个皮肤信息 |
| GET | `/api/scores/season-leaderboard` | 赛季排行榜（周重置） |
| POST | `/api/rift/start` | 开启大秘境（需钥石） |
| GET | `/api/rift/leaderboard` | 大秘境排行榜（层数/速度） |
| GET | `/api/rift/keystones/{playerId}` | 查询玩家钥石信息 |
| WS | `/ws/game` | 实时对战通信 |

### 技术栈

- **前端**：原生 Canvas API，单 HTML 文件，支持 PC/移动端
- **后端**：Java 8 + Spring Boot 2.7.18
- **通信**：REST API + WebSocket
- **存储**：内存 ConcurrentHashMap（可替换为 MySQL/Redis）
- **测试**：Playwright (Python) 自动化 AI 玩家

## 快速开始

```bash
# 1. 编译后端
cd server
mvn package -DskipTests

# 2. 启动服务器
java -jar target/king-of-survive-server-1.0.0-SNAPSHOT.jar

# 3. 打开浏览器
open http://localhost:8080
```

## AI 玩家模拟测试

```bash
# 8玩家×2模式全链路模拟
uv run python scripts/profile_simulator.py

# 8人多模式测试
uv run python scripts/multiplayer_test.py

# 留存率分析
uv run python scripts/retention_analyst.py
```

模拟器覆盖 8 种玩家画像（男/女 × 少年/青少年/青年/中年），每种画像有不同的操作风格和偏好，全流程测试注册→签到→皮肤→组队→游戏→结算→反馈。
